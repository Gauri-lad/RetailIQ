# 📈 RetailIQ – AI-Powered Sales Forecasting & Demand Intelligence

## Overview

RetailIQ is an end-to-end Business Intelligence dashboard built using Python and Streamlit. It analyzes retail sales data, forecasts future sales using machine learning and time series models, detects anomalies, and segments products based on demand.

## Features

- 📊 Interactive Sales Analysis Dashboard
- 📈 Time Series Forecasting (SARIMA, Prophet, XGBoost)
- 🚨 Sales Anomaly Detection
- 📦 Product Demand Segmentation
- 💡 Business Insights & Recommendations
- 🎛️ Interactive Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- XGBoost
- Prophet
- Statsmodels

## Dataset

Superstore Sales Dataset
